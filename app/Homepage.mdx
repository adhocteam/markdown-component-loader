## What else can it do?

Embed SVG directly into your mdx file, just as you would a jsx file:

```markdown
Here‚Äôs an inline SVG: {{<span className="emphasis">{'{'+'{'} {'<svg style={{ display: \'inline\', height: \'1em\'}} viewBox="0 0 304 290"><path fill="none" stroke="currentColor" strokeWidth="16" d="M2,111 h300 l-242.7,176.3 92.7,-285.3 92.7,285.3 z" /></svg>'} {'}'+'}'}</span>}}. You can see that nested curly braces are also fully supported.
```

Import functions, such as shared snippets or fancy library functions, using YAML front-matter:

```markdown
---
{{<span className="emphasis">{'imports:'}</span>}}
{{<span className="emphasis">{'  window: global/window'}</span>}}
---

Hello <code>{{<span className="emphasis">{'{{ window.navigator.userAgent }}'}</span>}}</code>
```

Import whole React components:

```markdown
---
{{<span className="emphasis">{'imports:'}</span>}}
{{<span className="emphasis">{'  Button: ui/button'}</span>}}
---

Please {{<span className="emphasis">{'{\{ <Button path="/login">Login</Button> }\}'}</span>}}
```

Destructured imports, with some added single quotes:

```markdown
---
{{<span className="emphasis">{'imports:'}</span>}}
{{<span className="emphasis">{'  \'{ Button, Panel }\': ui'}</span>}}
---

Hello <code>{{<span className="emphasis">{'{\{ <Button /> }\}'}</span>}}</code>
```
